Setup a NFS Server on Docker

Descrição do Projeto:

- Configuração de um servidor NFS/Docker, implementação de um plugin NFS para persistência de arquivos entre os nós do Swarm.
- Configuração do exports para os IPS dos nodes.
- Teste de ponto de montagem do NFS Share entre os nodes.
- Stack de deploy do compartilhamento NFS

Tecnologias utilizadas:

- Docker
- Swarm
- NFS
- Stack
- Linux
